# Blockquotes in Markdown
<!-- '>' symbol is used to create blockquotes in Markdown -->


## Basic Blockquote

> This is a simple blockquote.
<!-- Single '>' creates a blockquote -->


---

## Multiple Paragraphs Inside Blockquote

> This is the first paragraph inside the quote.
>
> This is the second paragraph inside the quote.
<!-- Empty quoted line creates separate paragraphs inside blockquote -->


---

## Nested Blockquotes

> This is the first paragraph inside the quote.
>
>> This is a nested paragraph inside quote.
<!-- Double '>>' creates nested blockquote -->

>>> This is deeply nested paragraph inside quote.
<!-- Triple '>>>' creates third-level nested blockquote -->

>
> This is another paragraph inside quote.
<!-- Returning to single '>' continues parent blockquote -->


---

## Bold and Italic Inside Blockquote

> This is **bold** and _italic_ inside quote.
<!-- Markdown formatting works normally inside blockquotes -->


---

## Headings Inside Blockquote

> # This is heading 1 inside quote.
> This is paragraph inside quote.
<!-- Headings can be used inside blockquotes -->


---

## Multiple Headings Inside Blockquote

> # This is heading 1 inside quote.
> ## This is heading 2 inside quote.
>
>> ### This is nested heading inside quote.
>> This is paragraph inside nested quote.
<!-- Nested blockquote can also contain headings and paragraphs -->

>
> ## This is another heading inside quote.


---
